다빈치리졸브는 유투브용 영상편집툴로 색보정기능, 강력한 편집, 무료라는 장점이 있다.



1. 설치 : 다빈치리졸브 검색 후 무료버전(왼쪽)으로 다운로드(무료로 충분) - 개인정보 전부 1 입력 후 다운 - 압축풀기 - 설치- 넥스크 반복 - 인스톨

1. 폴더정리하기
RECORD : 최상위 폴더
	VLOG : 프로젝트이름1
		DB : DB(빈칸으로 두기)
		EP1 : 첫번째 에피소드
			FOOTAGE : 촬영파일 넣기
			BGM : 음악파일 넣기
		EP2 : 두번째 에피소드
			...
	ENGLISH : 프로젝트이름2
		...
<중략>

1. 프로젝트시작하기 : 다빈치리졸브 켜기 - 익스플로러 아이콘 누르기 - 뉴 데이터베이스 - create- disk 타입 - 이름은 대문자X 띄어쓰기X 예) vlog_edit - location에서 db폴더 선택 - create 버튼 누름 - 익스플로러에서 db 생성 확인 - vlog_edit db 클릭 - 우측하단 new project 클릭 - ep01 에피소드 이름 입력(대소문자 띄어쓰기 가능) - create 버튼 누름 - 프로젝트 시작

1. 미디어 탭 정리하기 (첫번째 탭) : 냉장고에서 도마로 옮기는 과정. 냉장고에서 로케이션 지정 - 쓸 영상 파일 모두 선택해서 도마로 드래그앤 드롭 - 쓸 BGM 파일 모두 선택해서 도마로 끌고 오기 - 도마에서 우클릭하고 ADD BIN 선택 - FOOTAGE, BGM 폴더 생성 - 각각 나눠서 담아넣기

1. 에디트 탭 편집하기 (세번째 탭)
선택 : A + 클릭

자르기 : B + 클릭, 영상앞뒤 드래그

타임라인 확대/축소 : 알트 + 마우스휠

선택 영상 잘라내기 : 백스페이스

자막삽입 : 왼쪽 상단 Effects Library - Titles - Text나 Fusion Titles 클릭 - 우측 상단 인스펙터에서 자막 멘트 변경 - 폰트변경 - 크기 조절 - 밑으로 드래그해서 Drop Shadow에서 Offset을 22.000 정도로 그림자 조절 - Drop Shadow에서 Blur를 68 정도로 조정 - 투명도 Opacity를 80~100 사이로 조정 - Backgorund에서 width(약간), height(약간), opacity(80)를 지정해줘서 자막 Bar 생성 (필요시) - 완성된 자막 알트 + 자막 타임라인 이동해서 복사 - 멘트만 수정

비디오 트랜지션 : 왼쪽 상단 Effects Library - Cross Dissolve 선택(*그 외의 효과도 가능) - 영상과 영상 사이에 드래그앤드롭

볼륨조절 : 영상, 음악 선택 후 우측 상단 인스펙터에서 조정

블러(blur) / 투명도(opacity) 조절 : 영상, 자막 선택 후 우측 상단 인스펙터에서 조정

저장 : 최상단 메뉴에서 file - save project

1. 딜리버 탭 단축키 (마지막 탭) : 렌더 세팅즈 탭에서 유투브 1080p 선택 - fileName 지정 - 위치지정후 세이브 - Add to Render Queue 클릭 - 우측 큐에서 Start Render 클릭 - 영상제작 완료!
